# Modern Portfolio Website

A modern and responsive personal portfolio website built using HTML, CSS, and JavaScript.  
This project showcases skills, projects, and contact information with a clean UI and smooth animations.

---

## Features

- Responsive Design
- Smooth Scrolling Navigation
- Modern Dark Theme UI
- Animated Hero Section
- Skills Showcase
- Project Section
- Social Media Icons
- Mobile Friendly Layout

---

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome
